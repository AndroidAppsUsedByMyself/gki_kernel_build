# gki-kernel_build
## gki内核构建工具

## 使用方法:

1. 右上角Fork.

2. 修改配置文件`common_config.env` :

   |        Arguments        |                          Note                     |
   | :---------------------: | ------------------------------------------------- |
   |      KERNEL_SOURCE      | Your gki-kernel source repo                       |
   |  KERNEL_SOURCE_BRANCH   | Your gki-kernel source branch                     |

3. 点击`Actions`找到`All workflows`选择`gki-kernel-buildOS`点击`Run workflow`绿色按钮运行

## PS: 测试项目,报错自行解决.
